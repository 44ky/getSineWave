getSineWave.c
======================
It generates the sine wave (.wav) file.  
If you did want to set parameters, you could use options. 

Usage
------
    $ gcc getSineWave.c -o wave
    $ ./wave
And you'll get the wave file on current directory.  

Options
------
    $ ./wave -h
